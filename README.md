## 📖 About The Project

<img src="Article%20Preview/images/desktop.png" alt="Article Preview Component Desktop View" width="100%" />

The **Article Preview Component** is a minimal and modern UI element that displays an article summary along with author details and interactive share options.  
This component is perfect for **blogs, news portals, magazine websites, or UI practice projects** where users want to show article teasers in an elegant and compact way.

The design focuses on clarity, readability, and smooth interactions. By clicking the share icon, users can reveal a sleek popup containing social media sharing options — a valuable UI pattern commonly used in modern content-based websites.

This component is ideal for developers practicing **responsive design, layout structuring, and interactive UI behavior** using pure HTML, CSS, and JavaScript.

---

## ✨ Features

- Clean and modern UI  
- Responsive layout for all screen sizes  
- Interactive share popup with smooth animation  
- Author info section with image and details  
- Minimal, content-focused design  
- Fully customizable for any article or blog layout
